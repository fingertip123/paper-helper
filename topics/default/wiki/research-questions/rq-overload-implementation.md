# RQ: 行政超载如何影响政策执行绩效与地方国家能力？

## 问题陈述

资源相对职责的系统性不足（[[administrative-overload|官僚超载]]）通过什么机制削弱 [[policy-implementation|政策执行]] 和 [[state-capacity|地方国家能力]]？

## 现有工作如何回答

- [[dasgupta-2020]]：资源更少的官员执行更差，机制是 [[managerial-focus|管理焦点]] 被挤出；政治责任不清导致资源投入不足。

## 我的进路 / 假设

- 超载 → 管理焦点下降 → 执行赤字（两步机制）。
