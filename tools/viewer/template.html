🎁 导出 docx

将精心修改后的文档保存到本机文件夹；若文件夹不存在会自动创建。导出前会先保存编辑器中的未提交段落。

📎 插入引用

从知识库 source 页选取文献，插入 (作者, 年份) [[key]] 到当前光标位置。

🎨 界面主题

四套雅色主题，择一而驻；偏好将自动保存于本机。

🎀 偏好设置 · 大模型 API

新用户已默认配置智谱 GLM-4-Flash，可直接「纳入研究」与「知识查询」。也可自行替换 API Key 或其它免费模型。自定义 Key 仅保存在本机 .yanzhan/config.json。
